## ✨ Features

- Fixed frosted-glass navbar with CTA button
- Hero section with radial glow, pill badge & gradient text heading
- Trusted by logos strip
- 6-card features grid with icon boxes and hover lift effect
- Stats counter row (50K+ users, 99.9% uptime, 200+ integrations)
- 3-column testimonials with gradient avatars
- Pricing section — Starter / Pro (featured) / Enterprise
- CTA section with radial gradient glow
- Multi-column footer with social links

## 🧠 Advanced CSS Concepts Used

- `-webkit-background-clip: text` + `-webkit-text-fill-color: transparent` — **gradient text effect** on hero heading
- `box-shadow: 0 0 24px rgba(108,99,255,.4)` — **glowing button** effect
- `radial-gradient` — hero background glow + CTA section glow
- `backdrop-filter: blur(14px)` — frosted glass nav
- `transform: translateY(-4px)` — card hover lift
- `::before` pseudo-element — circular glow behind hero
- `repeat(auto-fit, minmax())` — responsive grid with no media queries
- `clamp()` — fluid typography
- `position: fixed` — navbar
- `scroll-behavior: smooth` + anchor links
- `border-color` highlight — featured pricing card
- `::before { content: '✓' }` — CSS-only feature list checkmarks
- Responsive breakpoints: `768px`, `480px`
